Analisei arquiteturas de aplicações Java e desenvolvi e mantive frameworks internos de persistência e Rich Internet Applications. Treinei novos engenheiros, conduzi reciclagens técnicas, revisei código de desenvolvedores juniores e participei de recrutamento e entrevistas técnicas.

## Engenheiro de Software Java Sênior · jan de 2013 - mai de 2014

Liderei migrações de servidores de aplicação de clientes do JBoss para GlassFish e monitorei a performance das novas rotinas e da infraestrutura. Treinei e supervisionei equipes em um framework baseado em JSF e desenvolvi funcionalidades e utilitários que simplificaram entregas e reduziram erros.

## Engenheiro de Software Java Pleno · set de 2011 - dez de 2012

Desenvolvi um framework JSF 2 e PrimeFaces que levou camadas e componentes reutilizáveis da plataforma Swing para o desenvolvimento web. Liderei a atualização de Java EE 5 para Java EE 6 e de JBoss 5 para GlassFish.

Projetei um mecanismo de atualização em larga escala para substituir Java Web Start, usando socket streams e um servidor central para monitorar diretórios e distribuir versões para filiais e terminais. Mantive frameworks internos de persistência para PostgreSQL e Oracle, um framework Swing e um ERP cliente/servidor com EJB, JMS e Quartz, além de contribuir com aplicações JSP e Spring.
