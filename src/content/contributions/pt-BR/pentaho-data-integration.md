## Minha contribuição

Adicionei a possibilidade de retornar os headers da resposta HTTP como campos de saída nos steps HTTP, HTTP POST e REST Client do Pentaho Data Integration.

A mudança incluiu metadados dos steps, interfaces de configuração, mensagens localizadas, comportamento em runtime e testes de integração.
